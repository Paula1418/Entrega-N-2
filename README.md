# Entrega-N-2
A partir del dataset comparando muestras de médula ósea (Bone Marrow) , Sangre perisferica PB y Leucemia, se realizaron distintas visualizaciones para analizar las diferencias en la expresión de marcadores celulares.

El gráfico de dispersión (FS vs SS) permitió observar la distribución celular según tamaño y granularidad, diferenciando claramente ambos tipos de muestra. Los boxplots mostraron la variabilidad de marcadores como CD34, CD13 y CD33, evidenciando diferencias en su nivel de expresión.

El catplot facilitó la comparación general de todos los marcadores, mientras que el pairplot permitió explorar correlaciones entre ellos y sus distribuciones. Finalmente, el lineplot con barras de error reflejó las tendencias y variabilidad de la expresión promedio por marcador, destacando aquellos con diferencias marcadas entre médula ósea y leucemia.

En conjunto, estas gráficas ofrecieron una visión clara y complementaria del comportamiento de los marcadores, ayudando a identificar patrones diferenciales de expresión entre las muestras.
